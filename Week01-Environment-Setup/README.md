# Week 1 – Cybersecurity Lab Environment Setup

## Objective:
Set up a safe virtual cybersecurity lab using VMware Workstation Pro, Kali Linux, and Ubuntu.

## Tools used:
* VMware Workstation Pro
* Kali Linux
* Ubuntu
* Terminal / Linux commands

## What I configured:
- Kali Linux virtual machine
- Ubuntu virtual machine
- NAT adapter for internet access
- Host-only adapter for isolated lab communication
- VMware Tools
- Clean Install snapshot

## Tests completed:
- Checked IP addresses using ip a
- Tested internet connectivity using ping google.com
- Tested Kali-to-Ubuntu communication
- Tested Ubuntu-to-Kali communication

## Reflection:
This lab helped me understand how virtual machines can be used to create a safe cybersecurity testing environment. I also learned the difference between internet connectivity through NAT and local communication through a Host-only network.
